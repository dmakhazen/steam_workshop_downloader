# SteamMod Manager

Одностраничный статичный сайт для управления списками модов Steam Workshop и генерации скриптов для SteamCMD.

## Возможности

- Выбор игры (RimWorld, Stardew Valley, Terraria)
- Просмотр Steam Workshop через iframe
- Альтернативный каталог модов прямо на странице (без iframe, через публичный прокси)
- Расширенные карточки каталога: превью, автор, рейтинг, подписчики, избранное, теги, описание
- Фильтры каталога: текст, тег, минимум подписчиков, сортировка по метрикам
- Фильтрация по сортировке, поиску, тегам и периоду
- Добавление модов по ссылке вида `...filedetails/?id=123456`
- Управление списками модов с сохранением в `localStorage`
- Экспорт/импорт списков в JSON
- Генерация скриптов SteamCMD: `.bat` и `.sh`
- Таблица альтернативных сайтов для скачивания модов

## Локальный запуск

Откройте `index.html` в браузере.

Для корректной работы удобнее использовать простой локальный сервер, например:

```powershell
python -m http.server 8080
```

После этого откройте `http://localhost:8080`.

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите файлы проекта.
2. Откройте **Settings → Pages**.
3. В **Build and deployment** выберите:
	- **Source**: `Deploy from a branch`
	- **Branch**: `main` (или `master`), папка `/ (root)`
4. Нажмите **Save**.
5. Через 1–2 минуты сайт станет доступен по ссылке вида:
	- `https://<username>.github.io/<repo-name>/`

## Установка SteamCMD

### Windows

1. Скачайте SteamCMD с официальной страницы Valve:
	- https://developer.valvesoftware.com/wiki/SteamCMD
2. Распакуйте в папку, например: `C:\SteamCMD`
3. Запустите `steamcmd.exe` один раз, чтобы SteamCMD обновился.
4. В приложении нажмите `⬇ .bat` для активного списка.
5. Поместите `.bat` рядом со `steamcmd.exe` или поправьте путь в скрипте.
6. Запустите скрипт двойным кликом.

### Linux

1. Установите SteamCMD по инструкции дистрибутива/Valve.
2. Убедитесь, что запускается `steamcmd.sh`.
3. В приложении нажмите `⬇ .sh`.
4. Сделайте скрипт исполняемым:

```bash
chmod +x your_list_name.sh
```

5. Запустите:

```bash
./your_list_name.sh
```

## Хранилище данных

Используются ключи `localStorage`:

- `steammod_lists`
- `steammod_active_list`
- `steammod_selected_game`

## Важно

- Из-за ограничений CORS содержимое Steam Workshop загружается в `iframe`.
- Взаимодействие с карточками модов внутри `iframe` выполняется вручную пользователем.
- Добавление модов в список выполняется через вставку ссылки на мод в сайдбаре.
- Для сценария без iframe доступен блок **"Каталог модов (без iframe)"**: он подгружает карточки через публичные прокси и позволяет добавлять моды кнопкой **"Добавить"**.
- Кнопка **"Обогатить карточки"** подтягивает данные со страниц модов (описание, теги, статистика), чтобы каталог был ближе к браузингу Steam Workshop.
- Публичные прокси могут быть недоступны или временно ограничены по лимитам — в таком случае повторите попытку позже.
